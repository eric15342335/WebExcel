## Description

Code generated by `GPT-4` and `Claude 3.5 Sonnet`.

Features:

* Cell Editing
* Load and Save CSV files
* Light and Dark Mode
* `LocalStorage` is used to store the data and theme settings
  * Data and theme settings won't be lost after closing the browser
* IE11 support
